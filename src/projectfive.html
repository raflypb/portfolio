Project

Aplikasi CMS dan Pendaftaran
Pondok Pesantren Tahfidzul Qur'an Al-Mabrur

Aplikasi ini dibuat berbasis Website sehingga dapat dijalankan menggunakan browser seperti Google Chrome, Mozilla Firefox, Microsoft Edge dan sejenisnya. Diperuntukkan pada Pondok Pesantren Tahfidzul Qur'an Al-Mabrur yang ada di Kota Ungaran untuk memudahkan proses penyebaran informasi beserta pendaftaran calon santri. Dibuat secara individu pada tahun 2023 dengan menggunakan HTML, CSS, PHP dan JavaScript serta memanfaatkan beberapa Framework seperti Bootstrap dan CodeIgniter 3.
